1. GAME OVERVIEW

    AirXonix is an ultra modern 3-dimensional remake of the famous Xonix game. In the Xonix game you have to rule a device, which is moving over the field with several monster-balls wandering inside. The objective is to cut the balls away from as much spare field as possible. The rules of AirXonix are almost the same, but now the device is flying in the air and everything is in full 3D! The stunning special effects are strengthened by 3D sound! New types of monsters and tons of different bonuses all made in full 3D bring the old game to an entirely new level. The complete AirXonix distribution includes 5 types of games with more then 80 levels. The rules are simple and the game will give unforgettable pleasure to the people of any age!

3. HOW TO PLAY

3.1 Game Rules

    You operate a flying device which moves above the surface of the game field. Your task is to fill as much of the game field as you can. To fill in the field, you fly above an area with no balls in it, being sure to not let the balls hit the line you are making until you have reached the other side of the empty space. Avoid contact with the mines, which move on filled areas. Each level has a time limit. Collect bonuses for extra lives, score, time and a bonus which slows down all balls and enemies.

3.2 Keyboard Controls:

Menu controls:

Up Arrow, Down Arrow Change selection
Enter or Space Choose and adjust an option
Escape or Backspace Return to a previous menu
Left Arrow, Right Arrow Change parameter (e.g. sound or music volume)

Default game controls:

Up Arrow Move forward
Down Arrow Move backward
Left Arrow Move left
Right Arrow Move right
   
Backspace Restart the current level
(available only 5 times per game)
   
Escape Abort game
Pause or P Pause game
Ctrl+M Change music
Ctrl+R Restart game
Ctrl+Q Quick exit to the main menu

3.3 Custom controls & Joystick

   If you want to use a Joystick enable the "Use Joystick" flag in the Engine Options dialog.

   Using the menu of the game options ("Controls" option) you can assign for each action a custom keyboard button, joystick button or joystick axis. Simply press the necessary button or turn the joystick handle on each required action.

3.4 Engine Options

   Push the "OPTIONS" button in start dialog to change engine options:

"OK" button - Save changes and exit.
"Cancel" button - Exit without saving.
"Reset to default" button - Reset to default values all parameters. In most cases it is the best choice.

"NEXT DEVICE" button - Select 3D display rendering device. You can choose the video adapter, in case if your system contains more than one adapter, and a hardware acceleration or program emulation mode.

"Video Mode" - Set display resolution in the game. Select the lower resolution to increase the performance or higher resolution to increase picture quality.

"High quality objects" - Disable this flag to increase the game performance (it is recommended for processors lower then 233 MHz or while the program works in software emulation mode).

"High quality textures" - Disable this flag to insignificantly increase the game performance on low-performance graphic accelerators or when running in software emulation mode.

"Use mip-mapping" - This option increases picture quality, with a minor penalty to graphics performance.

"Wait for vertical blank" - Synchronizes the rendering with the vertical blank status (VSYNC). On most computers enabling this flag greatly increases the picture quality. However, in the case of a low-performance graphic accelerator, disabling this flag can give better picture quality for some resolutions.

"Sound enable" - Disable this flag if sound and music are not needed. This will cause an insignificant increase in the game performance.

"Use Joystick" - Enables joystick support. This flag should not be enabled if you do not use a joystick.

"Joystick Sensitivity" - Select the sensitivity of the joystick axes. Experiment with it, and choose the most suitable variant for your system.
